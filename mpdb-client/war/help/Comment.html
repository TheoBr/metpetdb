Comment

Associated with a sample or a subsamples.

Required Attributes

Author

The user who wrote this comment. The user must be a registered user of the system (Member), but does not need to have upload privileges (Contributor etc.).

When

Date and time of when the comment was entered. Accuracy to the second is sufficient.

Message

The message text of the comment. This is a free form text field.
