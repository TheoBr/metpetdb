Headers for Upload Spreadsheets

Samples

More information about sample descriptors

REQUIRED FIELDS

  • SAMPLE NUMBER
  • ROCK TYPE
  • LATITUDE
  • LONGITUDE
  • REFERENCE : required for published data.

OPTIONAL FIELDS

  • LOCATION ERROR
  • IGSN
  • REGION
  • COUNTRY
  • ALIAS (alternate sample number)
  • COLLECTOR
  • DATE OF COLLECTION
  • PRESENT SAMPLE LOCATION
  • GRADE
  • COMMENT
  • MINERALS PRESENT (listed as mineral names)

Chemical Analyses

More information about chemical analysis descriptors

REQUIRED FIELDS

  • SAMPLE NUMBER
  • SUBSAMPLE
  • MINERAL (or whole rock)
  • METHOD
  • SUBSAMPLE TYPE : required to create subsamples at bulk upload
  • REFERENCE : required for published data.
  • ELEMENTS/SPECIES : at least one required
  • UNITS : read from beneath element/species
  • POINT

OPTIONAL FIELDS

  • ANALYTICAL FACILITY
  • ANALYSIS DATE
  • ANALYST
  • REFERENCE IMAGE
  • X STAGE
  • Y STAGE
  • X REFERENCE
  • Y REFERENCE
  • PRECISION : abs/rel read from beneath
  • TOTAL
  • COMMENT

Images

More information about image descriptors

REQUIRED FIELDS

  • SAMPLE NUMBER
  • PATH
  • IMAGE TYPE
  • ELEMENT : required for x-ray maps

OPTIONAL FIELDS

  • SUBSAMPLE
  • SCALE
  • COLLECTOR
  • DWELL TIME
  • CURRENT
  • VOLTAGE
  • COMMENT
