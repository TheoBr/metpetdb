Viewing Samples (private, shared, published, or public...)

In MetPetDB you may navigate to a table of samples of different ownership:

  • samples that you own and have uploaded yourself.
  • shared samples that are in a project to which you have permission.
  • published samples that are owned by the publication.
  • public samples, the owner of which has shared with the public.

Viewing a list of samples (or search results) is done in the same way.


The My Samples / Search Results Table

  • Attributes are listed for each sample
  • Samples can be sorted by a particular attribute by clicking on the header for that column
  • Column width can be adjusted by moving the mouse over the edge of the column and dragging left or right
  • The displayed attributes can be changed by clicking Custom view (This allows you to select which attributes are displayed).
  • In the sample table, links allow you to navigate to a particular sample or the owner profile, by clicking the sample name or owner name for a particular sample.
  • Complete sample information is displayed for a particular sample when the user clicks the "i" button.


The Sample Details Page

  • This simply lists the full attributes (including provenance) for a particular sample
  • Sample location is displayed on a map to the right.
  • Clicking on a thumbnail in images box allows you to navigate to image list.
  • If you own this sample you may:
    • Edit the attributes by clicking the edit button.
      • At this point you may add/remove images.
      • Click save when you are finished editing.
      • Note required attributes are shown with red stars.
    • Delete the sample.

Table of subsamples

  • The subsample table contains subsample attributes.
  • This may be sorted in the same way a sample list is sorted (see above).
  • A subsample may be added here by clicking Add Subsample at top right of list.
  • Links are also included:
    • Subsample detail page
    • Images page
    • Image map for this subsample (or create a new one)


The Subsample Details Page

  • This simply lists the full attributes for a particular subsample
  • A chemical analysis may be manually added here by clicking Add Chemical Analysis at top right of list.
  • Images (if any are uploaded) are shown as thumbnails in image box to right
    • Image attributes are shown beneath each image
  • Clicking on a thumbnail in images box allows you to navigate to image list
  • If you own this subsample you may:
    • Edit the subsample attributes by clicking the edit button.
      • At this point you may add/remove images.
      • Click save when you are finished editing.
      • Note required attributes are shown with red stars.
    • Delete the sample.

The Image Table

  • Thumbnails of each image for a particular subsample are displayed
  • Thumbnails may be clicked to go to view the full sized image
  • An image may be added manually (if you own the sample) by clicking Add Image.



Table of chemical analyses

  • The analyses table contains chemical analysis attributes.
  • This may be sorted in the same way a sample table is sorted (see above).
  • A link is also included to the chemical analysis detail page for a particular analysis.
  • Selected (checked) analyses can be deleted by the owner by clicking on remove.



The Chemical Analysis Details Page

  • This simply lists the full attributes for a particular analysis, including the weight%/ppm for particular elements/species.
  • Oxides and elements are displayed in this order.
  • The analysis can be edited or deleted by the owner by clicking on the edit or delete buttons.
