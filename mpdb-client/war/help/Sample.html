Sample

The basic unit of information is based on the physical sample, which is typically the hand sample collected in the field. (See page on general description of data types for a more general overview)

The types of information that will be stored for a sample include the following items.

  • The letter "R" means it is required.
  • The number preceding the data descriptor indicates the number of possibility entries. Note that required fields have one and only 1 entry.
  • The minimum amount of information about a sample is a unique sample number, location, and rock type.
