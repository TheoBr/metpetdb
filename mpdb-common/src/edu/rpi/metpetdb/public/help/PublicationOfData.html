Rules for Publication of existing data.

In Progress

In the future, we will need a way to convert working data to published data. This will require creating a "review copy" of the data that only the reviewers can examine. The review copy will be identical to the eventual published copy and will include all appropriate metadata about the samples.

When data that reside in the working database are published, a copy of the data should be made as the "published version", leaving the original version alone, but flagging the original version has having been published. The original version can still be edited by the owner. Rendering data as "published" will require system admin level access, and must be coordinated with the journal editors. We don't want people "publishing" data that hasn't been peer reviewed.
